# Images

Place your product and brand images here with the following filenames:

| File | Used In | Description |
|------|---------|-------------|
| `hero-bg.jpg` | Hero section background | Full-width product shot (e.g. floating matcha tins — image 1 or 3) |
| `product-midorinami.jpg` | Midorinami product card | Close-up of the Midorinami (緑波) tin |
| `product-hekido.jpg` | Hekido product card | Close-up of the Hekido (碧道) tin |
| `story-photo.jpg` | Our Story section | Lifestyle / tea ceremony photo |
| `gallery-1.jpg` | Scrolling gallery | Any product/lifestyle image (wide crop works best) |
| `gallery-2.jpg` | Scrolling gallery | Any product/lifestyle image |
| `gallery-3.jpg` | Scrolling gallery | Any product/lifestyle image |

## Recommended sizes
- `hero-bg.jpg` — 1920×1080px minimum
- Product images — 800×600px minimum, 4:3 ratio
- `story-photo.jpg` — 600×750px, portrait
- Gallery images — 840×560px minimum

The site degrades gracefully: each section shows a beautiful green-toned placeholder if an image file is missing.
